Change affiliate links TEAM GUIDE
How to point any country to a different offer/network. No coding, no rebuild — ~1 minute. Tracking stays intact automatically.

When to use this

  • Switch a country to a different network (e.g. FR → Blitz if OasisAds scrubs)
  • Update to a new offer / creative the network gives you
  • Point a country to any new checkout link

Steps (do this for each country you want to change)

1
Open the links file: github.com/francistucker5374/tryjetterix → assets/config.js (this link opens it in edit mode). Sign in if asked.
2
Scroll to the block titled "⚙️ AFFILIATE LINKS — EDIT HERE" (near the top).
3
Find the country line: us uk au de fr. Delete the // at the start of that line to switch it ON.
4
Paste the full new link between the quotes " ". Use the Direct-to-Checkout (DTC) link from the network. Do NOT add s1/s2 — we attach the jx-<geo> tracking tag automatically.
5
Click the green "Commit changes…" button (top right) → "Commit changes" in the popup.
6
Wait ~1 minute. That country now sends every "Buy" click to the new link. ✅ Done.
To undo: open the same file, put the // back in front of that country's line, commit. It returns to the default OasisAds link.
